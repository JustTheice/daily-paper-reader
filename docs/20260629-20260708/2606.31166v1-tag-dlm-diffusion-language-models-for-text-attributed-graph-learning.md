## 摘要
文本属性图（TAG）中每个节点都带有自然语言描述，要求模型能够同时对文本和图拓扑进行推理。现有方法通常分别处理两种模态：图神经网络处理浅层文本特征，而LLM与图的混合方法主要将语言模型用作文本编码器，并将结构学习委托给单独的图模块。我们提出的方法将文本推理和图消息传递统一在掩码扩散语言模型（一种具有双向注意力和生成式解码的语言模型）中。对于每个图实例，该方法将采样的局部邻域线性化为令牌序列，并通过拓扑注意力掩码注入图结构，从而实现图上的消息传递。由于扩散语言模型既能解释文本也能生成文本，因此该方法只需更改提示即可适应不同任务，支持节点分类、链接预测以及跨数据集迁移，无需针对特定目标进行微调。实验表明，该方法在两个任务的所有三个TAG基准测试中均优于图神经网络、图变换器和基于LLM的基线，最高比最强基线提升3.9个百分点。

## Abstract
Text-attributed graphs (TAGs), where each node carries a natural language description, require models to jointly reason over text and graph topology. Existing approaches often handle the two modalities separately: graph neural networks operate on shallow text features, while hybrids of LLMs and graphs use the language model mainly as a text encoder and delegate structure learning to a separate graph module. We propose method that unifies textual reasoning and graph message passing within a masked diffusion language model, a language model with bidirectional attention and generative decoding. For each graph instance, method linearises a sampled local neighbourhood into a token sequence and injects graph structure through a topology attention mask, which realises message passing over the graph. Because the diffusion language model can both interpret and generate text, the method adapts to different tasks simply by changing the prompt, supporting node classification, link prediction, and cross-dataset transfer with no target-specific fine-tuning.   Experiments show that method outperforms graph neural networks, graph transformers, and LLM-based baselines on all three TAG benchmarks across two tasks, improving over the strongest baseline by up to 3.9 points.

## 1. 核心问题与整体含义（研究动机和背景）

文本属性图（TAG）中每个节点附带自然语言描述，要求模型同时推理文本语义和图拓扑结构。现有方法通常将两种模态分离处理：
- **图神经网络（GNN）**：仅使用浅层词袋或截断嵌入表示文本，丢失大量语义信息。
- **LLM+图混合方法**：如LLaGA，将LLM作为冻结文本编码器，单独使用GNN在图结构上传播预计算的嵌入，两者缺乏交互，且任务特定输出头导致无法灵活迁移。

本文提出**TAG-DLM**，将**文本生成模型（掩码扩散语言模型，MDLM）**与**图结构学习**统一在同一框架中，使得文本理解和图推理在同一双向注意力过程中交互，并通过提示变更即可适应不同任务，无需任务特定微调。

## 2. 方法论

### 核心思想
将目标节点的局部邻域线性化为token序列，并通过**拓扑注意力掩码（Topology Attention Mask）**将图结构注入MDLM的注意力机制，使得单次前向传播同时完成文本编码和图消息传递。

### 关键技术细节
1. **子图线性化与提示构造**：
   - 对每个目标节点v，采样k跳ego图（默认k=2，每跳最多K=10个邻居）。
   - 线性化为序列：`target_token || neighbor1_token || ... || neighborN_token`。
   - 使用多项选择数字提示：答案位置设为`[M]`，候选答案映射为数字token（节点分类对应标签名，链接预测为“Yes/No”）。

2. **拓扑注意力掩码**：
   - 为每个样本构造一个二进制掩码矩阵`M_v`，控制哪些token span可以相互注意。
   - 默认采用**星型模式**：
     - 目标节点可注意所有节点（包括自身和所有邻居）。
     - 每个邻居节点仅注意自身和目标节点，不与其他邻居直接通信。
     - 填充位置双向屏蔽。
   - 优点：简单、任务无关、跨数据集稳定。

3. **骨干模型与微调**：
   - 使用**LLaDA-8B-Instruct**（MDLM，双向自注意力，生成式）。
   - 冻结基础权重，仅训练**LoRA适配器**（rank=64，alpha=64，目标模块all-linear）。
   - 损失函数：仅对答案位置`[M]`的交叉熵（邻居标签在训练时随机遮蔽防止标签泄露）。

### 理论支撑（Proposition 1）
在固定扩散时刻，拓扑掩码后的单层双向注意力等价于注意力消息传递（Attention-based MPNN），可恢复GAT为特例。证明了TAG-DLM无需独立GNN即可实现图推理。

## 6. 主要结论与发现

1. **全面最优**：在所有任务-数据集组合上，TAG-DLM准确率均超过最强基线（最高提升3.9个百分点）。
2. **链接预测优势更显著**：因为任务直接依赖关系证据，拓扑掩码的收益更大。
3. **跨数据集迁移有效**：特别是语义相似的源-目标对（如Cora→PubMed），且迁移能力归因于训练获得的图适应知识，而非仅依赖提示。
4. **拓扑掩码是关键**：移去或替换为ego-graph掩码均导致性能下降，星型掩码最适合目标中心预测。
5. **邻居数量影响**：K=10通常最优，太少缺乏证据，太多引入无关噪声。

## 7. 优点

- **统一框架**：无需任务特定输出头，通过提示即可处理NC和LP，甚至跨数据集迁移。
- **结构注入巧妙**：仅通过注意力掩码将图结构放入语言模型，无需修改骨干网络或添加额外GNN组件。
- **理论联系**：证明了拓扑掩码的注意力等价于注意力消息传递，为设计提供理论基础。
- **高效微调**：使用LoRA，冻结大模型大部分参数，训练开销可控。
- **生成式灵活性**：MDLM的双向注意力和生成能力自然适合图提示（顺序无关性，答案位置可填充）。
