## 摘要
离散扩散模型学到的是：一个去噪器、一个分数比，还是一个桥接插件预测器？在跳转率的层面上，这些是不同坐标下的同一个对象，而在错误的坐标下读取神经网络会改变所训练和采样的过程。从对任何加噪过程（包括边界项）的连续时间马尔可夫链（CTMC）ELBO的严格推导出发，我们证明了“预言距离”定理：负ELBO恰好等于数据熵加上从预言反向过程到学习过程的路径KL散度，而不仅仅是一个界限。因此，其唯一优化器是给定当前噪声状态时真实反向跳转率的条件期望，其不可约成本是前向过程$Z_t$破坏关于干净数据$Z_0$的信息的速率，即$-\tfrac{d}{dt}I(Z_0; Z_t)$，因此每个加噪过程共享相同的最佳可达负ELBO：数据熵。对于具有标记因子化噪声的序列，预言投影给出了优化器的三个精确坐标：去噪器、空穴（桥接插件）和分数，它们之间有闭式转换。该框架确定了文献中每个损失实际上优化的是哪个定律，将MDM、UDM、SEDD和GIDD作为特例恢复；解释了为什么去噪器和空穴在掩码扩散中一致但在均匀扩散中不一致；证明了去噪器参数化使均匀ELBO在初始化时发散，而桥接插件保持有限；并在初始化时精确校准ELBO实现。每个恒等式都在一个可精确求解的模型上数值验证，无需近似。

## Abstract
What does a discrete diffusion model learn: a denoiser, a score ratio, or a bridge plug-in predictor? At the level of jump rates, these are one object in different coordinates, and reading a neural network in the wrong coordinate changes the process being trained and sampled. Starting with a rigorous derivation of the continuous-time Markov chain (CTMC) ELBO for any noising process, boundary terms included, we prove the \emph{Oracle Distance} theorem: the negative ELBO is exactly equal to the data entropy plus the path KL from the oracle reverse process to the learned one, not merely a bound. Its unique optimizer is therefore the conditional expectation of the true reverse jump rate given the current noisy state, and its irreducible cost is the rate at which the forward process $Z_t$ destroys information about the clean data $Z_0$, $-\tfrac{d}{dt}I(Z_0; Z_t)$, so every noising process shares the same best achievable negative ELBO: the data entropy. For sequences with token-factorizing noise, the oracle projection yields three exact coordinates for the optimizer: denoiser, cavity (bridge plug-in), and score, with closed-form conversions among them. This framework identifies which law each loss in the literature actually optimizes, recovering MDM, UDM, SEDD, and GIDD as special cases; explains why denoiser and cavity coincide for masked diffusion but not for uniform diffusion; proves that a denoiser parameterization makes the uniform ELBO diverge at initialization while the bridge plug-in stays finite; and calibrates ELBO implementations exactly at initialization. Every identity is verified numerically, without approximation, on an exactly solvable model.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：离散扩散模型在实际应用（如文本等分类数据生成）中，学习到的反向过程通常被描述为多种看似不同的对象：去噪器、分数比、桥接插件预测器、留一法预测器等。这些术语常被当作同一模型的不同名称，但实际上它们**并不等价**：神经网络的输出在转换成跳转率之前不是一个反向过程，而在错误坐标下读取相同的输出会改变被优化和采样的连续时间马尔可夫链。因此，核心问题是：**负ELBO最小化到底是在优化什么？**
- **背景**：已有工作提出了多种参数化（MDM、UDM、SEDD、GIDD等），但缺乏统一的理论来解释它们的真实学习目标和数值行为差异（如均匀扩散中桥接插件稳定而纯去噪器发散）。本文旨在提供一个自洽的通用理论，揭示离散扩散模型学习目标的本质。

## 2. 方法论——核心思想、关键技术细节、公式说明

- **核心思想**：将连续时间马尔可夫链（CTMC）的变分下界（ELBO）解释为**路径空间上的相对熵**，而非仅仅是似然的一个上界。通过严格推导，证明负ELBO正好等于数据熵加上从Oracle反向过程到学习过程的路径KL散度（称为**Oracle Distance定理**）。由此，优化器是给定当前噪声状态下真实反向跳转率的条件期望（即边际反向率），而不可约代价是前向过程破坏信息率的积分。
- **关键技术细节**：
    - **CTMC ELBO的严格推导**：建立了基于Girsanov公式的路径散度公式，包含重建项、终端先验项和路径积分项。主要公式：
        \[
        \text{ELBO}_{[t_1,t_2]}(\theta;z_0) = \mathbb{E}_{q_{t_1|0}}[\log p^\theta_{0|t_1}(z_0|z_{t_1})] - \text{KL}(q_{t_2|0}(\cdot|z_0)\parallel p^\theta_{t_2}) - \int_{t_1}^{t_2} \mathbb{E}_{q_{t|0}}\left[ \sum_{y\neq z_t} \Phi(\hat{Q}_t(z_t,y|z_0),\hat{Q}^\theta_t(z_t,y)) \right]dt,
        \]
        其中 $\Phi(a,b)=a\log(a/b)-a+b$ 是局部速率散度。
    - **Oracle Distance定理（Theorem 3）**：平均负ELBO减去数据熵 $H(q_0)$ 等于重建KL加上路径KL（从Oracle反向过程到模型反向过程）：
        \[
        \text{NELBO}_{[t_1,t_2]}(\theta) - H(q_0) = \mathbb{E}_{q_{t_1}}[\text{KL}(q_{0|t_1}\parallel p^\theta_{0|t_1})] + \text{KL}(P^\star_{[t_1,t_2]}\parallel P^\theta_{[t_1,t_2]}).
        \]
    - **投影优化器**：模型率的最优解是清洁条件反向率的条件期望，即边际反向率 $\hat{Q}_t(z_t,y) = \mathbb{E}[\hat{Q}_t(Z_t,y|Z_0)|Z_t=z_t]$。
    - **速度不可约代价**：Oracle每时间代价 $J^\star_t = \frac{d}{dt} H(Z_0|Z_t) = -\frac{d}{dt} I(Z_0; Z_t)$，积分后等于数据熵。
    - **三种坐标体系**：对于标记因子化噪声，反向率可等价地表示为去噪器（$\pi^\star_i = q(z^i_0|z_t)$）、空穴/桥接插件（$\mu^\star_i = q(z^i_0|z^{-i}_t)$）和分数（$s^\star_i(z^i_t, y^i|z^{-i}_t) = q_t(y^i, z^{-i}_t)/q_t(z_t)$）。给出了闭式转换公式（Proposition 3, Table 2）。
    - **实际意义**：解释了掩码扩散中空穴与去噪器一致（因为掩码观测无信息），而均匀/GIDD中不一致；证明了无信息去噪器参数化会导致均匀扩散ELBO在初始化时发散（Proposition 6），而空穴参数化有限；给出了初始化标定恒等式 $\text{NELBO}_{\text{cavity, uniform}} = L \log V$。

## 6. 主要结论与发现

- **主要结论**：
    1. **离散扩散模型学习的是边际反向跳转率**，它是清洁条件反向率的条件期望，也是ELBO的唯一优化器。
    2. **负ELBO等于数据熵加上路径KL散度**（Oracle Distance定理），因此严格度量了模型反向过程与Oracle反向过程的距离。
    3. **所有加噪过程共享相同的最佳可达负ELBO：数据熵 $H(p_{\text{data}})$**，区别在于何时和如何花费。
    4. **反速率有三种等价坐标：去噪器、空穴、分数**，它们之间可通过闭式转换（Bayes更新）互相转换。误用坐标会导致系统性偏差（如将空穴头读作去噪器会产生额外KL损失）。
    5. **掩码扩散是特殊情况**：由于掩码标记不携带信息，空穴与去噪器一致（因此跨熵损失就是ELBO）。
    6. **去噪器参数化在均匀/GIDD扩散中导致ELBO在初始化时发散**，而空穴（桥接插件）参数化保持有限且可标定。
    7. **采样因子化误差**与反向率误差分离：即使Oracle零误差，因并行采样的因子化近似仍会产生性能损失（需较多采样步数才能减小）。
