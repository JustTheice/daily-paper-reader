## 摘要
离散扩散模型在质量上相对于自回归（AR）模型稳步提升。然而，这些模型通常受限于固定长度的生成，且不支持键值（KV）缓存。块扩散通过从左到右生成令牌块，部分弥合了扩散与AR之间的差距，但其固定大小的顺序块限制了解码的灵活性和并行性。在此，我们提出了一类新的语言模型——集合扩散，包括：(i) 一个将似然参数化分解为灵活位置、灵活长度令牌集合的因子化方法，以及 (ii) 一个支持每一步推理后更新KV缓存的集合因果扩散架构。通过将分解用于令牌集合而非固定大小的块，令牌可以以任意顺序的集合进行解码，包括滑动窗口集合，从而实现更快的推理并支持任意顺序解码。与先前的扩散语言模型相比，集合扩散在数学推理、摘要和无条件生成上取得了更好的速度-质量权衡，同时提供了比块扩散更强的填充性能。我们在项目页面上提供了代码、模型权重和博客文章：https://m-arriola.com/setdlms/

## Abstract
Discrete diffusion models have steadily improved in quality relative to autoregressive (AR) models. However, these models are normally constrained to fixed-length generation and do not support key-value (KV) caching. Block diffusion partially bridges diffusion and AR by generating token blocks left-to-right, but its fixed-size sequential blocks limit decoding flexibility and parallelism. Here, we present a new class of language models, set diffusion, comprised of (i) a likelihood parameterization that factorizes over flexible-position, flexible-length token sets and (ii) a set-causal diffusion architecture that supports KV cache updates after every inference step. By factorizing over token sets instead of fixed-size blocks, tokens can be decoded in arbitrarily-ordered sets, including sliding-window sets, enabling faster inference and support for any-order decoding. Set diffusion achieves better speed-quality tradeoffs on mathematical reasoning, summarization, and unconditional generation compared to prior diffusion language models while offering stronger infilling performance than block diffusion. We provide the code, along with the model weights and blog post on the project page: https://m-arriola.com/setdlms/

## 1. 核心问题与整体含义（研究动机与背景）

离散扩散语言模型（如 MDLM、SEDD）在生成质量上逐步接近自回归（AR）模型，但存在两个关键瓶颈：
- **固定长度生成**：大多数扩散模型无法输出可变长度序列。
- **缺少 KV 缓存**：推理时需对整个序列进行双向注意力计算，效率低下。

块扩散（BD3LM）通过**块级自回归 + 块内扩散**部分解决了上述问题，支持可变长度和 KV 缓存，但其固定大小、严格从左到右的块结构限制了解码的**灵活性**（如无法进行任意位置的填充生成）和**并行性**（每块必须完全解码后才能推进，且 KV 缓存只能在整块解码完成后更新）。

本文提出 **Set Diffusion（集合扩散）**，核心洞察是：**扩散与自回归之间的插值不应通过固定块大小实现，而应通过改变令牌集合的生成顺序来实现**。Set Diffusion 将似然分解为灵活位置、灵活长度的令牌集合，并设计集合因果扩散架构，支持每步推理后更新 KV 缓存，从而在速度、质量和灵活性之间取得更优权衡。

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想：令牌集合上的自回归分解
- 定义一个生成顺序 \(\sigma = (\sigma_1, \ldots, \sigma_N)\)，其中每个 \(\sigma_n\) 是非空、互不相交的位置子集，覆盖所有位置。
- 似然分解为：\(p_\theta(x) = \sum_\sigma \prod_{n=1}^N \pi(\sigma_n | x_{<\sigma_n}) \cdot p_\theta(x_{\sigma_n} | x_{<\sigma_n})\)，其中 \(\pi\) 是顺序策略，\(p_\theta\) 是令牌集合内的扩散模型。
- 通过 Jensen 不等式得到可训练的 NELBO：\(\mathcal{L}_{\text{SD}}(x;\theta) = \mathbb{E}_{\sigma\sim\pi} \sum_{n=1}^N \mathcal{L}(x_{\sigma_n}, x_{<\sigma_n};\theta)\)。

### 2.2 位置偏移调度（Position-Offset Schedules）：控制从左到右偏置
- 为每个位置 \(\ell\) 定义基于时间的激活区间 \([a_\ell, a_\ell + w]\)，其中 \(a_\ell = \frac{\ell-1}{L-1}(1-w)\)，\(w\) 是窗口大小。
- 在每个区间内，令牌被选中的概率呈单调递增（如线性或幂次形状）。
- \(w \to 1/L\) 时退化为纯自回归，\(w=1\) 时退化为顺序无关扩散（MDLM）。
- 通过调整 \(w\) 和形状参数 \(k\)，可平滑插值并控制期望推理预测预算 \(\bar{C}\)。

### 2.3 滑动窗口 SetDLM（SW-SetDLM）：实际实例化
- **训练**：使用单令牌集合（\(|\sigma_n|=1\)），采样完整排列后通过因果注意力一次性计算所有条件对数似然，梯度来自所有令牌，方差更低。
- **架构**：集合因果 Transformer，注意力掩码确保每个令牌只能看到前面集合的干净令牌及同集合的噪声令牌；支持每步更新 KV 缓存。
- **推理**：根据位置偏移调度选择候选位置（滑动窗口内），并行解码，接受高置信度令牌，立即更新 KV 缓存。

### 2.4 泛化性
Set Diffusion 统一了自回归（\(|\sigma_n|=1\)，固定左到右顺序）、顺序无关扩散（均匀 \(\pi\)）、块扩散（连续固定块）、任意顺序自回归（单令牌，均匀 \(\pi\)）和 FlexMDM（学习 \(\pi\)）等多种范式。
